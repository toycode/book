## 运算

您可以使用一些基本运算符进行数学运算，例如：
* **加法：** ```c = a + b```
* **减法：** ```c = a - b```
* **乘法：** ```c = a * b```
* **除法：** ```c = a / b```
* **取余：** ```c = a % b```

* **自加：** ```c = c + 5 或者 c += 5```
* **自减：** ```c = c - 5 或者 c -= 5```
* **自乘：** ```c = c * 5 或者 c *= 5```
* **自除：** ```c = c / 5 或者 c /= 5```
* **自取余：** ```c = c % 5 或者 c %= 5```
